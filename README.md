# Clima Tempo Simples

O <u>Clima Tempo Simples</u> é um projeto para atender as necessidades de uma agente de turismo da empresa Passe Bem.

### Pré-requisitos

* .NetFramework 4.6.1

* MS SQL Server 

### Como Executar
1. Execute os scripts contidos na pasta DB_Scripts na seguinte ordem: Create database, Create tables, Populate

2. Faça uma cópia do projeto através do comando  git clone ou fazendo o download do .zip do projeto.

3. Abra o projeto no editor escolhido e na seção  \<connectionStrings> do arquivo Web.Config localizado na pasta raiz do projeto e altere o data source para o nome do seu servidor.

4. Execute o projeto.
